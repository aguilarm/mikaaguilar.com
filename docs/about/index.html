Hello! I’m an experienced Software Engineer that primarily works in the Javascript ecosystem. Systems I build start with React and continuously deploy to Kubernetes.

Throughout my career, I’ve made it a point to have a varied experience. I believe every stack can offer valuable insights into problem-solving with software.

The incredible amount of excellent resources available to our industry is constantly inspiring, and I spend a lot of time reading new books and watching conference talks.

Simple and elegant solutions really get me excited. This site is primarily built on Hugo and I’m really hyped for things like Svelte. The JAMStack and the things companies like Vercel are up to are moving writing software for the web in an excellent direction.

As I’ve become more comfortable in my technical abilities I’ve started moving increasingly towards mentoring others.

Here are a few of the technologies I’ve worked with:

Frameworks & Libraries

I’ve built software on the foundations of things like NextJS, GraphQL, Golang, C#/.NET, and Drupal 8.

Tooling

To get projects up to prod, I’ve used CircleCI, Google Cloud Build and Ansible. To debug, I’ve looked to tools such as Grafana, Prometheus, DataDog, WeaveWorks and Blackfire.

Services

Production workloads I’ve managed have included systems such as NodeJS, Redis, Varnish, Apache Solr, PHP, Socket.io and many more.

Platforms

Most recently, I’ve been doing all of this on Google Cloud Platform but I’ve been deep in AWS and have used Digital Ocean for many a personal project. I would really love to be using some sort of serverless + static files driven solution for future projects.

With this site I hope to share insights I’ve collected along the way with you. Thanks for stopping by.
